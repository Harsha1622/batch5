Introduction to C

About C Language

C is a general-purpose programming language developed by Dennis Ritchie at Bell Laboratories in 1972. It is widely used for system programming, embedded systems, and application development because of its efficiency and flexibility.

Sample Program: Print Hello World


#include <stdio.h>

int main()
{
    printf("Hello, World!");
    return 0;
}

Explanation

#include <stdio.h> includes the standard input-output library.
main() is the entry point of the program.
printf() is used to display output on the screen.
return 0; indicates successful program execution.

Output

Hello, Worl
