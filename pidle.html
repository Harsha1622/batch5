Python IDLE & Program Editor

What is IDLE?

IDLE stands for Integrated Development and Learning Environment. It is a software tool provided with Python to write, run, debug and test Python programs.

Why IDLE is Used?

IDLE helps you write Python code, execute programs, find errors and view output easily in one place.

Parts of IDLE

IDLE has two main parts:

1. Shell – Used to execute small programs and see immediate results. It works in interactive mode.
2. Editor – Used to write, edit and save Python programs.

Python Shell (Interactive Mode)


>>> print("Hello")
Hello
Output is shown immediately after execution.

Program Editor

Program Editor is a place where you write Python programs, edit code, debug errors and save files. It is a text editor used to write Python source code.

Example Program (Editor Mode)


print("Welcome to Python")
Welcome to Python

Language Environment

Language environment helps you understand your code, run the program and view the output properly.

How to Check Python IDLE in Your System

Step 1: Click on Start button
Step 2: Select All Programs
Step 3: Click on Python
Step 4: Open IDLE
