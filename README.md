# SmartLearn (batch5)

A small static website for SmartLearn — course pages, dashboard, and profile templates.

Features
- Static HTML pages for course listings, syllabus, user signup, and profile.
- Lightweight and easy to host using GitHub Pages.

Quick start (view locally)
1. Clone the repository:
   git clone https://github.com/Harsha1622/batch5.git
2. Open `home.html` in your browser.

Changes applied in this commit
- Added consistent head metadata and accessibility attributes to all HTML files.
- Used inline styles (per request) to centralize presentation directly in HTML.
- Added LICENSE (MIT) and .gitignore.
- Improved README with deploy and contribution guidance.

Deploy (GitHub Pages)
1. In the repository Settings → Pages, set the source to the `main` branch and the root folder (/).
2. Wait a minute; the site will be available at:
   https://Harsha1622.github.io/batch5

Contributing
- Edit HTML files and keep styles inline if needed (project currently uses inline styles by request).
- Use semantic elements (header, nav, main, footer) and provide alt text for images.
