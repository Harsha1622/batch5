Introduction to Python

About Python Language

Python is a high-level, object-oriented, interpreted scripting language. A script is a set of instructions given to the computer, and Python executes these instructions line by line using an interpreter without the need for a compiler.

Programming Style

Python is a combination of both Procedural Oriented Programming (POP) and Object Oriented Programming (OOP). This makes Python flexible and easy to use for beginners as well as professionals.

History of Python

Python was developed by Guido van Rossum, a Dutch programmer. Development started in the late 1980s and early 1990s as a hobby project. Python was influenced by the ABC language and also borrowed features from languages like C, C++, and Perl. The name Python was inspired by the comedy TV show Monty Python's Flying Circus.

Sample Program: Print Hello World


print("Hello, World!")

Explanation

print() is a built-in function used to display output.
Python does not require header files or a main function.
The program is executed directly by the Python interpreter line by line.

Output

Hello, World!

Features of Python

✔ Easy to learn and read using simple English words.
✔ Open source and free to use.
✔ Portable – write once, run anywhere.
✔ Interpreted language.
✔ Dynamically typed (no need to mention data type explicitly).
✔ Requires fewer lines of code compared to C, C++ and Java.

Applications of Python

✔ Web Development using Django, Flask and FastAPI.
✔ Artificial Intelligence and Machine Learning.
✔ Data Science and Data Analysis.
✔ Automation and scripting tasks.
✔ Mobile App Development using Kivy and BeeWare.
✔ Game Development using Pygame.
✔ Desktop applications using Tkinter, PyQt and wxPython.

File Extension

Python programs are saved using the .py extension.
