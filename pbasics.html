Python Basics

Identifiers

Identifiers are names used to identify variables, functions, classes, modules, and objects in Python. 


total_score = 85
user_name = "admin"
calculate_total()

Rules for Identifiers:
✔ Can contain letters, digits, and underscore (_).
✔ Must start with a letter or underscore.
✔ Cannot start with a digit.
✔ Special symbols are not allowed except underscore.
✔ Keywords cannot be used as identifiers. 


2total        # ❌ Invalid
total_2       # ✅ Valid
total@value   # ❌ Invalid

Keywords

Keywords are reserved words in Python that have predefined meanings. They cannot be used as variable names or function names.

All keywords are written in lowercase except True, False, and None. 


if user_logged_in:
    print("Access granted")

Variables

Variables are used to store data values. Python does not require explicit variable declaration. The data type is automatically determined by the value. 


user_id = 1024
username = "developer"
account_balance = 2500.75

Rules for Variables:
✔ Variables are case-sensitive.
✔ Must start with a letter or underscore.
✔ Cannot start with a number.
✔ Special characters are not allowed except underscore.
✔ No limit on variable name length. 


status = "active"
print(status)   # ✅ Correct
print(Status)   # ❌ Error

Indentation

Indentation refers to spaces at the beginning of a line. Python uses indentation instead of curly braces to define code blocks. By default, Python uses 4 spaces. 


if account_balance > 0:
    print("Transaction allowed")
    print("Processing payment")

Comments

Comments are used to explain code logic. They improve readability and are ignored by the Python interpreter.

1. Single-line Comment


# Display welcome message
print("Welcome to SmartLearn")

2. Multi-line Comment


'''
This program checks user access
and displays a message accordingly
'''
print("Access granted")
