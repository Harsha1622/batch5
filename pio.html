Input and Output Statements in Python

Output Statement

Output statements are used to display information on the screen. In Python, the print() function is used to produce output.

Sample Program: Display Output


print("Welcome to Python")
print(10 + 20)

Explanation

print() displays text or values on the screen.
It can print strings, numbers, and expressions.
Multiple outputs can be displayed using multiple print statements.

Output

Welcome to Python
30

Input Statement

Input statements are used to take data from the user. Python uses the input() function to accept input. The input is always treated as a string.

Sample Program: User Input


name = input("Enter your name: ")
print(name)

Formatted Input and Output

Formatted input and output are used to display user data in a clear and readable format.

Sample Program: Formatted Output


name = input("Enter your name: ")
age = int(input("Enter your age: "))

print(f"Name: {name}")
print(f"Age: {age}")

Output

Name: Alex
Age: 21
