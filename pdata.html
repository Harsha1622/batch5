Data Types in Python

What is a Data Type?

A data type specifies the type of value a variable can hold. Python automatically determines the data type based on the value assigned to a variable.

Classification of Data Types

Python data types are classified into the following categories:

✔ Numeric Data Types
✔ Sequence Data Types
✔ Set Data Types
✔ Mapping Data Type
✔ Boolean Data Type
✔ None Data Type

1. Numeric Data Types

Numeric data types are used to store numerical values. Examples include int, float, and complex. 


a = 10
b = 3.5
c = 2 + 3j

print(type(a))
print(type(b))
print(type(c))
<class 'int'>
<class 'float'>
<class 'complex'>

2. Sequence Data Types

Sequence data types store multiple values in an ordered manner. Examples include list, tuple, and string. 


lst = [1, 2, 3]
tup = (10, 20, 30)
name = "Python"

print(type(lst))
print(type(tup))
print(type(name))
<class 'list'>
<class 'tuple'>
<class 'str'>

3. Set Data Types

Set data types store unordered and unique values. The main example is set. 


s = {1, 2, 3, 3}
print(s)
print(type(s))
{1, 2, 3}
<class 'set'>

4. Mapping Data Type

Mapping data type stores values in key-value pairs. The example is dictionary. 


student = {"name": "Harsha", "age": 20}
print(type(student))
<class 'dict'>

5. Boolean Data Type

Boolean data type represents two values: True or False. 


x = True
print(type(x))
<class 'bool'>

6. None Data Type

The None data type represents the absence of a value. 


value = None
print(type(value))
<class 'NoneType'>
